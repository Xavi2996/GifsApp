GifsApp

Aplicación web desarrollada con Angular que permite buscar y visualizar
GIFs utilizando la API de Giphy. El proyecto implementa funcionalidades
modernas como scroll infinito, caché de búsquedas y consumo de APIs
REST.

------------------------------------------------------------------------

🚀 Características

-   🔎 Búsqueda de GIFs
-   🔥 Sección de Trending GIFs
-   ♾️ Scroll infinito para cargar más resultados
-   💾 Caché de búsquedas para optimizar peticiones
-   📱 Interfaz responsive (implementada con CSS Grid)
-   ⚡ Consumo de API REST

------------------------------------------------------------------------

🛠️ Tecnologías utilizadas

-   Angular 19
-   TypeScript
-   HTML5
-   CSS3
-   RxJS
-   Giphy API

⚙️ Instalación

1.  Clonar el repositorio

git clone https://github.com/Xavi2996/GifsApp.git

2.  Entrar al proyecto

cd GifsApp

3.  Instalar dependencias

npm install

4.  Ejecutar la aplicación

ng serve

5.  Abrir en el navegador

http://localhost:4200
